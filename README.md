# ☕ Coffee Feedback Web

Projeto acadêmico desenvolvido com **Node.js** e **Express**, cujo objetivo é criar um servidor HTTP capaz de processar dados enviados por um formulário HTML.  
A aplicação simula um sistema simples de **feedback de uma cafeteria**, permitindo que usuários enviem avaliações sobre produtos consumidos.

---

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido como parte de uma atividade acadêmica da disciplina **Programação para Internet**, com o objetivo de:

- Criar um servidor HTTP utilizando Express
- Processar dados enviados por formulários HTML
- Trabalhar com requisições do tipo POST
- Retornar respostas dinâmicas ao usuário
- Publicar a aplicação na plataforma Vercel

---

## 🛠️ Tecnologias Utilizadas

- Node.js
- Express.js
- HTML5
- CSS3
- Vercel (deploy)
- GitHub (versionamento)

---

## 📋 Funcionalidades

- Exibição de formulário de feedback da cafeteria
- Envio de dados via formulário HTML
- Processamento das informações no servidor
- Resposta personalizada de acordo com a nota informada
- Interface simples e intuitiva
